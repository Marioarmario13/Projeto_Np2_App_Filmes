# App Filmes 🎬

Aplicativo Android desenvolvido para recomendação de filmes, com interface interativa, sinopses detalhadas e uma experiência simples de navegação entre telas.

## 📱 Funcionalidades

- Tela de login simulada
- Tela inicial com um carrossel de imagens de filmes
- Visualização de sinopses de filmes como:
  - 007
  - Capitão América
  - Chico Bento
  - Nosferatu
- Navegação entre telas via `Intent`

## 🧱 Arquitetura e Tecnologias

- **Linguagem:** Java e XML
